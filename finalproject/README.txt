Lunk “Adventure” Puzzle Game

Press Run to play.

Player will be brought to a menu where they have the option of choosing between basic control instructions or starting a new game.

Player controls - left, right, up and down buttons for movement. Spacebar to jump.

New game will bring player to cutscene - Intro cutscene can be advanced by pressing the ‘f’ button.

Once finished, the player can choose which puzzle they would like using ‘1’, ‘2’, or ‘3’ button.

‘r’ will return the player to the puzzle select screen.

More specific instructions are given with each puzzle.
